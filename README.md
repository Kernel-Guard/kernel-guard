## Flagship Projects

### Aegis-BPF
A runtime security engine built with eBPF CO-RE and LSM-based enforcement, designed for synchronous kernel-level protection and production-focused observability.

### Post-Quantum Messaging App
A secure communication project exploring post-quantum cryptography and next-generation messaging system design.

### CathodeX
An AI-driven platform for accelerating battery material discovery through machine learning and graph-based modeling.
